PEFF UniProt Fetcher

Generate a PEFF file for any organism directly in your browser. Runs entirely client-side via Pyodide — no server involved.


 e.g. 83333 (E. coli K-12), 9606 (human), 559292 (yeast)
Heads up: Large proteomes (e.g. human, mouse) can be slow and memory-heavy in a browser tab — expect several minutes and hundreds of MB of RAM. For big jobs, use the CLI instead.
